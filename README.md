# MOVEAM B2B — 제품 단가 포털

업체(거래처) 사장님들이 로그인해서 제품 사진·단가를 열람하고, 우리가 관리자 화면에서 단가를 직접 관리하는 B2B 단가표 서비스.

## 핵심 기능

- **업체용**: 브랜드별 제품 목록 → 제품 검색 → 사진 + 도매/소매 단가 열람 → 카톡/전화 1:1 문의 버튼
- **관리자용**: 제품 등록/수정/삭제, 사진 업로드, 단가 즉시 반영, 업체 계정 발급/차단
- **브랜드 확장형**: 번웨이(BurnWay)로 시작, HKS·IRP 등 여러 브랜드로 확장

## 기술 스택

| 영역 | 선택 |
|------|------|
| 프론트 | Vite + React + Tailwind CSS |
| 백엔드 | Supabase (Auth + Postgres + Storage) |
| 배포 | GitHub Pages (`gh-pages`) |

## 왜 이 구조인가

- 도매 단가는 로그인한 업체만 열람 (Supabase Auth + RLS)
- 서버 관리 불필요 — 정적 호스팅 + BaaS, 전부 무료 플랜
- pos-calculator-web과 동일 패턴이라 운영 노하우 재사용
